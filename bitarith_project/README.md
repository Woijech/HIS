# BitArith (ЛР: двоичные коды, IEEE-754, BCD 5421)

Реализация арифметики на **битовых массивах (0/1) длиной 32**:
- Прямой / обратный / дополнительный коды (целые)
- Сложение в дополнительном коде (ввод в 10-ом)
- Вычитание через `A + (-B)` (доп. код)
- Умножение и деление в прямом коде (деление: точность 5 знаков)
- IEEE-754-2008 Single (32-bit): `+ - * /`, конвертация из десятичной строки
- BCD 5421 (32 бита = 8 цифр): кодирование/декодирование/сложение
