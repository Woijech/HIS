# Лабораторная работа №2  
## Построение СКНФ и СДНФ на основании таблиц истинности

Проект реализован на **Python 3** и использует **только стандартную библиотеку**.  
Сторонние библиотеки для решения основной задачи не применяются.

## Что умеет программа

- принимать логическую функцию в свободном формате;
- поддерживать операции `!`, `&`, `|`, `->`, `~`;
- поддерживать также Unicode-аналоги: `¬`, `∧`, `∨`, `→`, `↔`, `≡`;
- работать с переменными `a`, `b`, `c`, `d`, `e`;
- строить таблицу истинности;
- строить СДНФ и СКНФ;
- выводить числовую форму СДНФ и СКНФ;
- выводить индексную форму функции;
- определять принадлежность функции классам Поста;
- строить полином Жегалкина;
- находить фиктивные переменные;
- вычислять булевы производные: частные и смешанные (до 4 переменных);
- минимизировать функцию:
  - расчетным методом;
  - расчетно-табличным методом;
  - табличным методом (карта Карно).

### 1. Запуск программы

Из корня проекта:

```bash
python -m logiclab
```

Или через отдельный файл запуска:

```bash
python run.py
```

После запуска откроется обычное консольное меню с пунктами управления.
Сценарий работы:
- один раз ввести логическую функцию;
- сразу перейти в меню операций;
- последовательно вызывать нужные пункты без перезапуска программы;
- в любой момент заменить функцию или выйти.

### 2. Запуск тестов

```bash
python -m unittest discover -s tests -v
```

### 3. Проверка покрытия

```bash
coverage run -m unittest discover -s tests -v
coverage report -m
```

## Как пользоваться меню

После запуска программа сразу просит ввести логическую функцию.
На этапе ввода можно:
- ввести выражение для анализа;
- ввести `h` для справки;
- ввести `0` для выхода.

После ввода открывается меню операций для выбранной функции:
- таблицу истинности;
- СДНФ и СКНФ;
- числовые формы;
- индексную форму;
- классы Поста;
- полином Жегалкина;
- фиктивные переменные;
- булевы производные;
- минимизацию расчетным методом;
- минимизацию расчетно-табличным методом;
- минимизацию табличным методом;
- краткую сводку и полный отчет.

Пользователь может:
1. Ввести функцию один раз.
2. Сразу работать с меню операций.
3. Вызывать несколько пунктов подряд для той же функции.
4. Ввести новую функцию прямо из меню операций или завершить программу.

## Пример входной функции

```text
!(!a->!b)|c
```

## Что выводит программа

- переменные и их порядок;
- таблицу истинности;
- СДНФ и СКНФ;
- числовые формы;
- индекс функции;
- классы Поста;
- полином Жегалкина;
- фиктивные переменные;
- булевы производные;
- все стадии склеивания;
- таблицу простых импликант;
- карту Карно;
- итоговую минимизированную ДНФ.

## Ограничения

- количество переменных: от 1 до 5;
- карта Карно для 5 переменных строится в виде двух слоев 4x4;
- булевы производные выводятся для комбинаций от 1 до 4 переменных.
